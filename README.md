# MayCloud
2018

Cloud Computing Graduation Project

The purpose of the project is to allow users to store their personal files in a virtual environment easily and securely without using any space on their computers and allow users to access them at any time and place through internet.
Created by using PHP, CSS, JavaScript, MySQL.
